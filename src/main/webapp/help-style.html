Specifies the graph style of the plot. This can be any one of:
  • Area - a filled area graph. (area)
  • Bar - a simple 2D bar graph. (bar)
  • Bar 3D - a 3D styled bar graph. (bar3d)
  • Line - a line graph with indicated data points. (line)
  • Line simple - a line graph without indicated data points. (lineSimple)
  • Line 3D - (line3d)
  • Stacked Area - an area graph with each plot stacked . (stackedArea)
  • Stacked Bar. (stackedBar)
  • Stacked Bar 3D. (stackedBar3d)
  • Waterfall. (waterfall)
